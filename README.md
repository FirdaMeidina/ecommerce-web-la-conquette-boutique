# E-commerce Web La Conquette Boutique

[![Status](https://img.shields.io/badge/Status-Development-blue)]()

## Deskripsi Proyek
**La Conquette** adalah butik e-commerce yang menghadirkan koleksi eksklusif **pakaian, sepatu, dan rok** yang memancarkan **pesona, keanggunan, dan keberanian**. Terinspirasi dari gaya *"conquette"* yang chic, feminin, dan modern, setiap produk dirancang dengan detail dan kualitas terbaik untuk menemani pelanggan menonjol di setiap kesempatan.

Website ini dibuat untuk memberikan pengalaman belanja online yang **mudah, interaktif, dan stylish**, membantu La Conquette menjangkau pelanggan lebih luas secara digital.

## Fitur Utama
- **Katalog Produk Lengkap**: Menampilkan gambar berkualitas tinggi, deskripsi, harga, ukuran, dan kategori produk.  
- **Keranjang Belanja & Checkout**: Tambah, hapus, dan beli produk dengan mudah.  
- **Pencarian & Filter Produk**: Cari produk berdasarkan kategori, harga, atau popularitas.  
- **Dashboard Admin**: Kelola produk, kategori, stok, dan data penjualan.  
- **Responsif**: Tampilan optimal di desktop, tablet, dan smartphone.  
- **Integrasi Pembayaran (opsional)**: Mendukung gateway pembayaran online.  

## Teknologi yang Digunakan
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap  
- **Backend:** PHP 
- **Database:** MySQL 
- **Version Control:** Git / GitHub  

## Instalasi
1. Clone repository:
   ```bash
   git clone https://github.com/username/ecommerce-web-la-conquette-boutique.git


2. Import database MySQL dari file `database.sql`.
3. Sesuaikan konfigurasi database di `config.php` atau `.env`.
4. Jalankan server lokal (XAMPP, WAMP, Laravel Valet, atau lainnya).
5. Buka website melalui browser:

   ```
   http://localhost/ecommerce-web-la-conquette-boutique
   ```

## Cara Penggunaan

1. Login sebagai **Admin** atau **Pelanggan**.
2. **Admin:** Tambah, edit, dan hapus produk serta kategori.
3. **Pelanggan:** Jelajahi katalog, tambahkan produk ke keranjang, dan lakukan pembelian.
